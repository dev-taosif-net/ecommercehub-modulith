# Getting Started: Prerequisites

## Required Tools

| Tool | Version | Download |
|---|---|---|
| .NET SDK | 10.0 or later | https://dotnet.microsoft.com/download |
| JetBrains Rider | Latest | https://www.jetbrains.com/rider/ |
| Git | Latest | https://git-scm.com/ |

## Verify Installation

```powershell
dotnet --version   # Should output 10.x.x
```

## Optional Tools

- **Docker Desktop** — for running databases locally (PostgreSQL, SQL Server)
- **Bruno / Postman** — for testing API endpoints
